# Sobre mim

Atualmente atuo no desenvolvimento de soluções Full Stack, com foco em APIs REST, integrações, automações e aplicações web.

Tenho experiência no desenvolvimento de aplicações utilizando **Node.js**, **Java**, **React**, **PostgreSQL** e **Docker**, além de criar fluxos automatizados com **n8n**.

Também possuo experiência com infraestrutura Linux, administração de VPS, suporte técnico, CRM e análise de dados.

Estou sempre buscando evoluir como desenvolvedor e construir soluções escaláveis, performáticas e bem estruturadas.
